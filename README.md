NeoVim
====

### Depends

- NeoVim
- Python
- Haskell

### Setup
  
`
$ git clone 'https://github.com/eliz40x/nvim.git' "$HOME/.config/nvim" ; sh "$HOME/.config/nvim/setup.sh" 
`
